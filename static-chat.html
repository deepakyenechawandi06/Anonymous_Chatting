Anonymous Chat

Server Connection Required

This application requires a WebSocket server to function properly.

To run the server:

  1. Open Command Prompt (not PowerShell)
  2. Navigate to this folder: 
    cd C:\Users\pruthvi\OneDrive\Pictures\Chat
  3. Install dependencies: 
    npm install
  4. Start the server: 
    npm start
  5. Once the server is running, open index.html in your browser

If you're seeing a blank page in index.html, it's likely because the WebSocket server isn't running.
